# 📖 Qix Reader — Revisi

Aplikasi **e-reader** atau pembaca dokumen digital dengan berbagai fitur. Versi revisi dari aplikasi Qix Reader sebelumnya.

## ✨ Fitur

- **Baca Dokumen** — Mendukung format dokumen digital
- **Navigasi Halaman** — Fitur scroll dan navigasi halaman
- **Responsif** — Tampilan yang nyaman untuk membaca

## 🛠️ Tech Stack

- **HTML5** — Struktur halaman
- **CSS3** — Styling
- **JavaScript** — Interaktivitas

## 🚀 Cara Menjalankan

Buka file `index.html` atau akses melalui web server.
